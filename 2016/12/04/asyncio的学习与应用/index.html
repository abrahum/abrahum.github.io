asyncio的学习与应用

发表于 2016-12-04 更新于 2021-02-04 分类于 code

一直在制作完善pixiv的爬虫，最近遇到了一个问题，学校的网络环境太差了，导致了IO阻塞时间过长，常常出现线程长时间阻塞，占用了时间和内存。

为了解决这个问题，我决定尝试制定一个超时的图片删除重新爬取的过程，三次爬取不全则log报错不再爬取。

同时将整个脚本搬运至3.5版本，重新用asyncio异步编写，减少线程使用。
