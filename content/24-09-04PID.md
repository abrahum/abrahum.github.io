Piping Instriment Diagram 简称 PID，中文名：工艺管道和仪表流程图

<!-- more -->

## 仪表标识

| 首位  |   含义    |
| :---: | :-------: |
|   A   |   分析    |
|   D   |   密度    |
|   F   |   流量    |
|   L   | 物位/液位 |
|   P   |   压力    |
|  PD   |   压差    |
|   T   |   温度    |

| 末位  |    含义    |
| :---: | :--------: |
|   E   |  一次元件  |
|   G   | 视窗、观察 |
|   I   |   指示器   |
|   T   |   变送器   |
|   Q   |    记录    |

举例：

- PIT：带远传压力指示器
- PG：就地压力表
- TI-TE：热电偶温度指示器
